# Cardily Website

A modern, responsive website for Cardily - a student-led nonprofit dedicated to making cardiology more accessible to the next generation.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Sticky navbar with smooth scrolling
- **Interactive Elements**: Hover effects and scroll animations
- **Accessible**: Semantic HTML and proper ARIA labels

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations
- `README.md` - This file

## Getting Started

Simply open `index.html` in a web browser to view the website. No build process or dependencies required.

## Sections

1. **Hero Section** - Introduction and call-to-action
2. **Mission Section** - About Cardily and core values
3. **What We Do** - Features and services
4. **Vision Section** - Long-term goals
5. **Get Involved** - Contact and participation options
6. **Footer** - Links and information

## Customization

The website uses CSS custom properties (variables) defined in `styles.css` for easy theming. You can modify colors, spacing, and other design elements by updating the `:root` variables.

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
